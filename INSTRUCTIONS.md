# Инструкция: Как собрать данные

## 1. Сделайте фотографии

Сделайте 100-200 фотографий человека.

Важно:
- Разные ракурсы (спереди, сбоку)
- Разное освещение (день, вечер)
- Разная одежда (2-3 комплекта)
- Разное расстояние (лицо, весь человек)

Пример:
- 50 фото: лицо крупно
- 50 фото: человек полностью
- 50 фото: в разных позах
- 50 фото: в разных местах

## 2. Разметьте фотографии

### Способ A: Roboflow (проще)
1. Зайдите на https://app.roboflow.com
2. Создайте проект
3. Загрузите фото
4. Рисуйте прямоугольники вокруг человека
5. Экспортируйте в формате YOLO

### Способ B: LabelImg (локально)
1. Установите: pip install labelImg
2. Запустите: labelImg
3. Откройте папку с фото
4. Нажимайте 'w' для рисования прямоугольника
5. Назовите класс (например, "person")

## 3. Формат файлов

После разметки должно быть:
- Фотографии: datasets/images/train/ и datasets/images/val/
- Разметка: datasets/labels/train/ и datasets/labels/val/

Каждый файл разметки (.txt) содержит:

0 0.5 0.5 0.3 0.4

Где:
- 0 - номер класса
- 0.5 - X центра (0-1)
- 0.5 - Y центра (0-1)
- 0.3 - ширина (0-1)
- 0.4 - высота (0-1)

## 4. Сколько нужно фотографий?

Минимум:
- 50 фото для тестирования идеи
- 100 фото для работающего прототипа
- 200+ фото для хорошей модели
