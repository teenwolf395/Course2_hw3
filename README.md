# Проект YOLOv5 для распознавания участника корейской поп-группы enhypen NI-KI

## Структура папок:
- datasets/ - данные для обучения
  - images/ - фотографии
    - train/ - для обучения (80%)
    - val/ - для проверки (20%)
    - test/ - для тестирования
  - labels/ - разметка (файлы .txt)
- models/ - сохраненные модели
- test_videos/ - видео для тестирования (как развития проекта)
- results/ - результаты работы
- scripts/ - наши скрипты

## Точность определения:
 Данная обученная нейросеть является прототипом, где в обучении использовались для train 93 изображения, для validation 20 изображений.
 В итоге получено: Точность (precision): 0.783, Полнота (recall): 0.636.

## Вывод 
    Нейросеть при распознавании участника группы ошибается, если: 
    -закрыта большая часть лица 
    -человек в одежде похожей на ту, в которой был участник при обучении
    -изображение подвергалось сильной обработке
    -изображение темное и нечеткое
    Для улучшения распознавания рекомендуется:
    - увеличить количество изображений для обучения
    - можно сконцентрировать нейросеть на распознавание конкретно лица этого участника
    - использовать версию нейросети с более высокой точностью
    - для ускорения обучения использовать GPU
